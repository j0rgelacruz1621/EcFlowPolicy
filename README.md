# EcFlowPolicy

Sistema de emisión de pólizas de seguros y control histórico de clientes.

## Descripción

EcFlowPolicy es una aplicación diseñada para gestionar la emisión de pólizas de seguros y llevar un control histórico de los clientes. El objetivo principal es facilitar el proceso de aseguramiento y crear un historial confiable para la administración y consulta de clientes a lo largo del tiempo.

## Características

- Emisión de pólizas de seguros
- Registro y gestión de clientes
- Control histórico de acciones y cambios
- Interfaz sencilla y amigable
- Impresion de pilizas.
- Control de ventas.
